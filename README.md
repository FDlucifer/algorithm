## 推荐阅读

### 刷题必备

- 《剑指offer》
- 《编程之美》
- 《编程之法:面试和算法心得》  
- 《算法谜题》 都是思维题

### 基础

- 《编程珠玑》Programming Pearls
- 《编程珠玑(续)》
- 《数据结构与算法分析》
- 《Algorithms》 这本近千页的书只有6章,其中四章分别是排序，查找，图，字符串，足见介绍细致

### 算法设计

- 《算法设计与分析基础》
- 《算法引论》 告诉你如何创造算法
- 《Algorithm Design Manual》算法设计手册 红皮书
- 《算法导论》 是一本对算法介绍比较全面的经典书籍
